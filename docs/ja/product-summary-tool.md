# Product summary tool

English: [Product summary tool](../product-summary-tool.md)

複数の bench:product result を集計する tool です。by mode, by task, by mode/task の表を作ります。

## Key points

- npm run bench:product-summary
- --out で Markdown report を出力
- extension I/O と session I/O を表示
- run directory または actual-results.json を入力にできる
