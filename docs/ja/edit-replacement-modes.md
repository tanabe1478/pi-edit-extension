# Edit replacement modes

English: [Edit replacement modes](../edit-replacement-modes.md)

bench:product で使う edit replacement mode を定義します。built-in edit だけを外し、read/write/bash は残します。

## Key points

- pi_edit: built-in read/edit/write/bash
- replace_edit_tagged: tagged edit path
- replace_edit_hashline: hashline range path
- replace_edit_hybrid: hashline first, tagged fallback
- replace_edit_policy: recommended routing policy
